# Employee SQL Project

## 📌 Project Overview
This project focuses on analyzing employee and department data using SQL.

## 🛠️ Tools Used
- SQL (MySQL/PostgreSQL)

## 📊 Key Features
- Created relational tables (Employees, Departments)
- Used JOINs to combine data from multiple tables
- Applied aggregate functions like AVG, SUM
- Wrote subqueries for advanced analysis

## 🔍 Key Queries
- Highest paid employee in each department
- Average salary by department
- Employee count per department
